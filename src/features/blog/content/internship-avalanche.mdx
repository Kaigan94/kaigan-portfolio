## A New Chapter Begins

I'm thrilled to announce that I've started my 6-month internship (LIA) at Avalanche Studios Group! This is an incredible opportunity to work on real game projects and learn from an amazing team of developers.

## What I'll Be Doing

As a **UI Developer**, I'll be working on game development projects, focusing on creating intuitive and engaging user interfaces that enhance the player experience. The role combines both creative design thinking and technical implementation.

## Learning & Growth

During this internship, I'll be expanding my skill set across multiple areas:

### Building on Existing Skills

- **TypeScript/JavaScript**: Deepening my expertise in modern JavaScript development
- **React**: Applying React patterns in a game development context

### New Technologies

- **Perforce**: Learning industry-standard version control for game development
- **C++**: Diving into game engine development and performance-critical code
- **Python**: Exploring scripting and tooling for game development workflows

## The Team & Culture

Everyone at Avalanche Studios Group has been incredibly welcoming, helpful, and supportive. The collaborative environment makes it easy to learn, ask questions, and grow as a developer.

## Looking Forward

I'm excited to:

- Contribute to a real game project
- Learn from experienced game developers
- Bridge my web development background with game UI development
- Grow both technically and professionally
